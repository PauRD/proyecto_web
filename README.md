# Instalación y ejecución:

Sigue estos pasos para poner el proyecto en marcha:

1. **Clona el repositorio**

```bash
git clone https://github.com/PauRD/proyecto_web.git
cd proyecto_web
```

2. **Despliega los contenedores:** Ejecuta el siguiente comando en la raíz del proyecto:

```bash
docker-compose up -d
```

3. **Accede a la aplicación:** Una vez que los contenedores estén corriendo, abre tu navegador y visita: http://localhost:8080
